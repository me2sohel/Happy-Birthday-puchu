## 🎨 Customization Guide

### Change the Birthday Person's Name
Edit `index.html` and find this line:
```html
<h1 class="title">🎂 Happy Birthday Puchu! 🎂</h1>
```
Replace `Puchu` with your desired name.

### Change Colors
Edit `style.css` to customize the gradient background:
```css
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

Popular color combinations:
- Purple to Pink: `#667eea` → `#f093fb`
- Blue to Cyan: `#0066ff` → `#00ccff`
- Orange to Red: `#ff6b35` → `#f7931e`

### Add Custom Messages
In `index.html`, duplicate this block and edit the text:
```html
<div class="message">
    <p>"Your custom message here! 🎈"</p>
</div>
```

### Change Balloon Colors
In `style.css`, modify the color values for each balloon class:
```css
.balloon-red { background: #ff6b6b; }
.balloon-blue { background: #4ecdc4; }
/* etc. */
```
