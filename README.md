# 🎂 Happy Birthday Puchu! 🎂

A beautiful, interactive birthday celebration website created with HTML, CSS, and JavaScript.

## Features ✨

- 🎉 **Confetti Animation** - Animated confetti that falls when you click the button
- 🎈 **Floating Balloons** - Colorful balloons that float across the screen
- 🎂 **Animated Cake** - A cute cake with a flickering candle
- 🎊 **Beautiful Gradient Background** - Smooth color transitions
- 📱 **Fully Responsive** - Works on desktop, tablet, and mobile devices
- ✨ **Sound Effects** - Fun celebratory beeps
- 💬 **Personalized Messages** - Birthday wishes and messages
- 🌟 **Smooth Animations** - Professional CSS animations throughout

## How to Use 🚀

1. **Open the website**: Simply open `index.html` in your web browser
2. **Click the button**: Click the "Click Me!" button to trigger confetti and celebrate
3. **Share it**: Deploy to GitHub Pages or any web hosting service

## Customization 🎨

### Change the Name
Edit `index.html` and replace "Puchu" with the birthday person's name:
```html
<h1 class="title">🎂 Happy Birthday [Name]! 🎂</h1>
```

### Change Colors
Edit `style.css` to change the gradient colors:
```css
body {
    background: linear-gradient(135deg, #YOUR_COLOR1 0%, #YOUR_COLOR2 100%);
}
```

### Add Custom Messages
Edit the messages section in `index.html`:
```html
<div class="message">
    <p>"Your custom message here! 🎈"</p>
</div>
```

## Deploy to GitHub Pages 🌐

1. Your repository is already set up!
2. Go to Settings → Pages
3. Select `main` (or `master`) branch as source
4. Your site will be live at: `https://me2sohel.github.io/Happy-Birthday-puchu`
